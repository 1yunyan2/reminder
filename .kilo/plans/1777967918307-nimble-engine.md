以下是你相对于 ESP-IDF 5.3.4 (ESP32-S3) **出厂默认值**实际修改过的选项。
共分为 6 个 menuconfig 菜单路径。

## 二、你通过 idf_component.yml 引入的外部组件

这些不是 menuconfig 选项，但它们会**自动注册 menuconfig 菜单项**（如 LVGL、ESP-SR）。

```yaml
# main/idf_component.yml
dependencies:
  espressif/esp-sr: "^2.3.1"           # 语音识别（提供 ESP Speech Recognition 菜单）
  espressif/esp_codec_dev: "^1.5.4"     # 音频 Codec 设备驱动
  espressif/esp_audio_codec: ^2.4.0     # 音频编解码器
  espressif/esp_websocket_client: ^1.6.0 # WebSocket 客户端
  espressif/servo: ^0.1.0              # 舵机驱动（LEDC PWM）
  lvgl/lvgl: ^9.5.0                     # LVGL 图形库（提供 LVGL configuration 菜单）
  espressif/esp_lvgl_port: ^2.7.0       # LVGL ESP-IDF 移植层
```

> **注意**：`espressif/servo` 组件会自动在 menuconfig 中注册 `Audio Codec Device Configuration` 相关的
> Codec 支持选项（如 ES8311）。你选中的 `CONFIG_CODEC_ES8311_SUPPORT=y` 是该组件的默认值，无需手动改。

## 四、移植步骤（按顺序执行）

### 方法 A：直接复制（推荐，最简单）

```bash
# 1. 创建新项目
idf.py create-project my_new_project
cd my_new_project

# 2. 从旧项目复制以下文件（覆盖）
copy <旧项目>\sdkconfig .
copy <旧项目>\sdkconfig.defaults .
copy <旧项目>\partitions.csv .
copy <旧项目>\dependencies.lock .

# 3. 修改根 CMakeLists.txt，在 project() 行后添加：
#    spiffs_create_partition_image(assets gif FLASH_IN_PROJECT)

# 4. 复制组件依赖
copy <旧项目>\main\idf_component.yml main\

# 5. 复制源码和资源
xcopy <旧项目>\main\bsp\ main\bsp\
xcopy <旧项目>\main\ui\ main\ui\
copy <旧项目>\main\main.c main\
copy <旧项目>\main\CMakeLists.txt main\
xcopy <旧项目>\gif\ gif\

# 6. 编译（自动下载 managed_components）
idf.py build
```

### 方法 B：手动 menuconfig（适合理解配置）

```bash
# 1. 创建新项目，复制 idf_component.yml
# 2. 运行 idf.py menuconfig，按上面"一"中的表格逐项设置
# 3. 复制 partitions.csv、源码、gif 资源
# 4. 修改 CMakeLists.txt
# 5. idf.py build
```

## 五、重要提示

1. **`sdkconfig` 是自动生成的** — 它包含了所有 2895 行配置（包括芯片自动检测值）。你真正需要关注的只有上面列出的 **约 15 个手动更改项**。
2. **如果目标也是 ESP32-S3 + 相同 Flash/PSRAM 硬件**，直接复制 `sdkconfig` 最省事。
3. **如果硬件不同**，删掉 `sdkconfig`，只保留 `sdkconfig.defaults` + 手动 menuconfig 设置。
4. **`managed_components/` 不需要复制** — `idf.py build` 会根据 `idf_component.yml` 自动下载。但复制它可以加速构建。
5. **`CONFIG_LV_CONF_SKIP=y`** 意味着 LVGL 不读取 `lv_conf.h`，所有 LVGL 配置都在 menuconfig 中完成。
